## Descripción

Este portfolio está desarrollado en **HTML, CSS y JavaScript vanilla**, con un enfoque en diseño limpio y animaciones suaves para mejorar la experiencia visual del usuario. Incluye:

- Carrusel de imágenes en la sección HERO.
- Nombre "MIKAN" con animación flotante y efecto de resplandor en la sección ABOUT.
- Sparkles dinámicos en HERO y ABOUT generados con JavaScript.
- Scroll suave con botón "BACK TO TOP".
- Parallax simple en la foto de ABOUT.
- Diseño totalmente responsivo y optimizado para pantallas de diferentes tamaños.

## Tecnologías

- **CSS3**
- **JavaScript (Vanilla JS)**

## Animaciones y efectos

- **Carrusel automático** con flechas de navegación y transición suave.
- **Nombre "MIKAN"** con animación de flotación y resplandor dinámico.
- **Separador ABOUT** con resplandor suave animado.
- **Estrellas dinámicas** en HERO y sección ABOUT generadas con JS.
- **Hover effects** en flechas, enlaces y iconos de skills.
- **Scroll Parallax** en la foto de ABOUT.

> Nota: Aunque GSAP está cargado, actualmente todas las animaciones funcionan con CSS y JS vanilla.

## Instalación

1. Clonar o descargar este repositorio.
2. Abrir cualquiera de los archivos HTML (`index.html`, `works.html`, `contact.html`) en un navegador moderno.
3. Asegurarse de que la carpeta `assets` esté completa para que las imágenes, íconos y estilos funcionen correctamente.
