## Theme Draft

The project showcases Micaela Lucía Arce Gutiérrez’s **creative work in visual design, illustration, UX/UI, motion graphics, and 3D**. It emphasizes a **modern, interactive, and visually striking experience**, using subtle motion effects, glowing accents, and floating sparkles to create a dynamic, memorable portfolio.
