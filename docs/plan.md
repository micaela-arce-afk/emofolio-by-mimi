# Plan del Proyecto - Portfolio Angel Mikan

## 1. Objetivo del proyecto

El objetivo principal de este proyecto es crear un **portfolio web personal** para mostrar los trabajos de Micaela Lucía Arce Gutiérrez en **diseño visual, ilustración, UX/UI, Motion Graphics y 3D**. Se busca combinar estética, funcionalidad y experiencia de usuario fluida.

## 2. Alcance

- Diseño y desarrollo de **3 páginas principales**: `index.html`, `works.html`, `contact.html`.
- Implementación de **animaciones suaves y efectos visuales** (carrusel, resplandor de texto, estrellas flotantes).
- Diseño **responsive** para diferentes tamaños de pantalla.
- Preparación de documentación del proyecto (README, LICENSE, brief, inspiración, etc.).
- Optimización de imágenes y carga rápida (formato `.webp` y lazy loading).

## 3. Herramientas y tecnologías

- **HTML5 / CSS3 / JS Vanilla**
- **Fuentes**: Google Fonts (Poppins)
- **Animaciones**: CSS (transiciones, keyframes) y posible uso de GSAP para efectos opcionales.
- **Imágenes y recursos**: `.webp` optimizado, iconos `.webp` para skills.
- **Versionado**: Git y GitHub.

## 4. Entregables

- Portfolio funcional con navegación entre páginas.
- Animaciones de carrusel, resplandor de texto y estrellas flotantes.
- Documentación completa (`README.md`, `LICENSE`, `plan.md`, `project-brief.md`, `project-inspiration.md`, `project.yaml`).
- Archivos optimizados para web (imágenes, CSS, JS).

## 5. Cronograma tentativo

| Fase                 | Tareas                                  | Tiempo estimado |
| -------------------- | --------------------------------------- | --------------- |
| Diseño conceptual    | Bocetos, moodboard, colores, tipografía | 1 semana        |
| Maquetación HTML/CSS | Estructura de páginas, responsive       | 1 semana        |
| Funcionalidad JS     | Carrusel, scroll, botones, animaciones  | 1 semana        |
| Integración final    | Optimización, pruebas, correcciones     | 3-4 días        |
| Documentación        | README, LICENSE, briefs, inspiración    | 2-3 días        |

## 6. Criterios de éxito

- El portfolio debe ser **visualmente atractivo y funcional**.
- Carga rápida y experiencia de usuario fluida.
- Todas las páginas deben ser **responsive** y accesibles.
- Documentación completa y clara.
